Модуль 11. Условие (до изменения):

Написать приложение для работы с REST сервисом http://fecore.net.ua/rest/. Реализовать следующий функционал:

  1. функция getUsers должна получать текущий список всех пользователей в БД.
  2. функция addUser должна записывать в БД юзера с полями name и score.
  3. функция removeUser должна удалять из БД юзера по id.
  4. функция updateUser должна обновлять данные пользователя по id.
