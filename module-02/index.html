Модуль 2 - Домашнее задание

  1. Есть три курорта: taba, sharm или hurgada.
  2. Необходимо через prompt попросить ввести число, от 1 до 3-х.
  3. В тексте prompt необходимо написать какое число соотвествует какому курорту, на ваш выбор.
  4. После этого вывести сообщение о том, что пользователь выбрал такой-то курорт или сообщение о том, что введено число, варианта курорта для которого не было.

Push F12 :)
